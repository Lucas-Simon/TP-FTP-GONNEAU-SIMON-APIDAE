Lancer le serveur (Serveur.java) puis le client (Client.java).
Rentrer l'adresse IP puis le port.
Une fois connecté :

- retr monFichier.txt (exemple: retr test.txt), le fichier que l'on veut récupérer du serveur doit être dans le dossier ftp.
- stor monFichier.txt (exemple: sotr test.txt), le fichier que l'on veut envoyer sur le serveur doit être à la racine du dossier.
- dele monFichier.txt (exemple: dele test.txt), supprimer un fichier du dossier ftp.
